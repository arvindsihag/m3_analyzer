## 🎯 Overview

The **Action-Engagement-Collaboration (AEC) Triad** is a novel framework that integrates multimodal signals to analyze and evaluate human-robot collaboration. This repository contains the official implementation of our HRI 2026 paper.

## ✨ Features
- 🎥 **Multimodal Analysis** - Video, pose, gaze, and speech modalities
- 🤝 **Collaboration Metrics** - Action, Engagement, and Collaboration scores
- 📊 **Interactive Visualization** - Monitoring collaboration quality
- 📈 **Benchmarking Tools** - Standardized evaluation on HRC datasets
